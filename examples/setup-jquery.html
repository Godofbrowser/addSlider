addSlider: jQuery Setup

Similarly to the JavaScript Method, you can setup an element as a slider using the jQuery plugin $(selector).addSlider(settings).

For information on how to set the initial value or get the value read the Readme or view the HTML Setup example.

Example:

Create your <input> element using HTML:

<input id="slider1" />

Now after the DOM is ready call the jQuery plugin:

$("#slider1").addSlider();
