addSlider: JavaScript / jQuery Setting

For the JavaScript Setup method pass in a key-value-pair object as the the second argument. For the jQuery Setup method pass in a key-value-pair object as the first argument.

For the full list of settings read the Readme or view the HTML Settings example.

Example:

<input id="slider1" value="200,800" />

$add.Slider("#slider1", {
  min: 100,
  max: 1000,
  step: 25,
  range: true,
  formatter: function(x){return "$"+x}
});


Alternativly the same can be accomplished with jQuery:

$("#slider1").addSlider({
  min: 100,
  max: 1000,
  step: 25,
  range: true,
  formatter: function(x){return "$"+x}
});
